# AI Safety Audit

A structured audit of AI model safety, covering jailbreak testing, bias evaluation, and guardrail proposals.

## Structure

- `report/` — Final audit report (PDF)
- `jailbreak-tests/` — Prompts and results from jailbreak testing
- `bias-tests/` — Prompts, results, and images from bias evaluation
- `safety-framework/` — Proposed guardrails and safety recommendations
